#VerifyLocal user research templates for councils about to start service transformation

24 Nov 2016

These templates were developed for the Verify pilots discovery workshop for user research, the aim of which was to provide local authorities with practical methods and tools to conduct user research.

Interview prompt - a sample script for user researchers to use when conducting user research interviews

 

User journey mapping template - a suggested method for user researchers to gather and record their findings, mapping out step-by-step the user's experience of the service

 

Consent form - agreement for subjects of user research to sign
